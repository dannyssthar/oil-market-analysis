# Oil Market Analysis

A data-driven exploration of global oil market dynamics using Python.

This project analyzes historical oil price data to uncover trends, volatility patterns, and macroeconomic insights relevant to energy markets.

---

## Objectives

- Clean and process real-world oil price datasets
- Perform exploratory data analysis (EDA)
- Visualize trends and structural changes
- Extract actionable economic insights

---

## Tech Stack used for this project

- Python
- Pandas
- NumPy
- Matplotlib

## Sample Insights

- Price volatility clustering observed during geopolitical shocks
- Long-term cyclical patterns identifiable
- Structural regime changes across decades

---

This extension incorporates US Treasury yield data (FRED)
to examine how interest rate movements influence oil returns.

Pipeline includes:

- API data ingestion
- Time-series alignment
- Log-return transformation
- Correlation visualization
- OLS factor regression

All scripts are documented for reproducibility and learning.
