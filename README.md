# Oil Market Analysis

A data-driven exploration of global oil market dynamics using Python.

This project analyzes historical oil price data to uncover trends, volatility patterns, and macroeconomic insights relevant to energy markets.

---

## Objectives

- Clean and process real-world oil price datasets
- Perform exploratory data analysis (EDA)
- Visualize trends and structural changes
- Extract actionable economic insights

---

## Tech Stack used for this project

- Python
- Pandas
- NumPy
- Matplotlib
